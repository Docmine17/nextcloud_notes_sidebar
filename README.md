# Nextcloud Notes Sidebar

Uma extensao para o Mozilla Firefox que permite acessar suas notas do Nextcloud rapidamente atraves da barra lateral do navegador.

## Funcionalidades

* Acesso rapido: Suas notas ficam a um clique de distancia na barra lateral do Firefox.

## Como instalar (Modo de Desenvolvimento)

1. Abra o Firefox e digite "about:debugging" na barra de enderecos.
2. No menu lateral, clique em "This Firefox".
3. Clique no botao "Load Temporary Add-on...".
4. Navegue ate a pasta deste projeto e selecione o arquivo "manifest.json".
5. A extensao sera carregada. Clique no icone nas configuracoes da barra de ferramentas ou no menu lateral para acessar.

## Tecnologias Utilizadas

* Vanilla JavaScript
* HTML5 / CSS3
* Firefox WebExtensions API (Manifest V3)
